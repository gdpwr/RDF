# Governance Model – Renewable Data Framework (RDF)

## Convener (Founding Organization)
- GoodPower (Roger Lopez, Chief Growth & Data Officer).
- Maintains initial repo, publishes v0.1 → v1.0.

## Working Group
- Multi-organization collaborative body (NGOs, researchers, policymakers).
- Reviews drafts, approves schema updates, manages roadmap.

## Adopters & Contributors
- Open to anyone via pull requests.
- Public comment process for schema drafts.

## Release Cycle
- Draft → Public Comment → Working Group Approval → Release.
- Minor releases every 6 months, major releases every 18–24 months.
- Semantic versioning (1.0.0, 1.1.0, 2.0.0).
